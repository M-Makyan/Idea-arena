🔥 Connect Firebase

Idea Arena needs a free Firebase Realtime Database to sync across all your devices. It takes 3 minutes to set up once.

  1. 1Go to console.firebase.google.com → create a project (disable Analytics to save time)
  2. 2Click Build → Realtime Database in the left sidebar → Create database → pick any region → Start in test mode → Enable
  3. 3You'll see your database URL at the top — it looks like https://your-project-default-rtdb.firebaseio.com — copy it
  4. 4⚠️ If you get a permission error: go to Rules tab in your database and set both read and write to true, then Publish
